# KP Reader VI — Worked Examples from the Book

All examples from K.S. Krishnamurti's "KP Reader VI: Horary Astrology" with their horary numbers, procedure, and verified answers.

## Method Types
KP Horary uses TWO methods to determine the ascendant:
1. **Horary Number (1-249)**: Querist gives a number, which maps to a fixed ascendant via the 249-sub table
2. **Chart Cast for Moment**: Ascendant calculated from the exact date/time/location the question was asked (like a natal chart)

Examples 1, 2, 7, 13, 14 use method 2 (moment chart). All others use method 1 (horary number).

## Role of Date/Time/Location
- **Planet positions**: Calculated for the judgment moment (essential for significator analysis)
- **House cusps**: Placidus system requires latitude
- **Day Lord**: Day of the week → ruling planet (e.g., Sunday=Sun, Tuesday=Mars)
- **Moon position**: Determines Moon sign lord and Moon star lord for ruling planets

## Key Patterns from the Book

### The Decision Formula
1. Find the **primary cusp** for the question category
2. Find the **sub-lord** of that cusp using the 249-sub table
3. Check: Is the sub-lord in **direct motion**? (retrograde = weakens/denies)
4. Check: Is the sub-lord in constellation of a **direct planet**? (retrograde constellation lord = denies)
5. Find what houses the **constellation lord of the sub-lord** signifies
6. If constellation lord signifies favorable houses → **YES**
7. If constellation lord signifies unfavorable houses → **NO**

### Important: Different Questions Check Different Cusps
| Question | Primary Cusp | Book Also Checks |
|----------|-------------|-----------------|
| Marriage | 7th | 2nd, 11th |
| Job/Career | 10th | 2nd, 6th |
| Finance/Money | 2nd | 6th, 11th |
| Children | 5th | 2nd, 11th |
| Health/Cure | 1st (recovery), 11th (cure) | 6th, 8th |
| Foreign Travel | **12th** | 3rd, 9th |
| Property | 4th | 11th, 12th |
| Legal | 6th | 11th |
| Education | 4th | 9th |
| Promotion | 10th | 2nd, 6th, 11th |
| Missing Person | 2nd, 7th, 11th | — |
| Imprisonment | **12th** | — |
| Loan Repay | **12th** | — |

### Critical: Some questions use 12th cusp, not the "obvious" cusp
- Foreign travel → 12th cusp (not 9th)
- Imprisonment → 12th cusp
- Loan repayment → 12th cusp (expenses going out)
- Hospitalization → 12th cusp
